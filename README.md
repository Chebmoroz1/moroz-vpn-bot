# VPN Bot для Telegram

Telegram бот для управления VPN сервером (AmneziaWG) с возможностью создания и управления VPN ключами.

## Возможности

- 🔐 Авторизация пользователей по номеру телефона
- 🔑 Создание VPN ключей (AmneziaWG)
- 📋 Просмотр списка ключей пользователя
- 🗑 Удаление VPN ключей
- ⚙️ Админ-панель (базовая версия)
- 📱 Генерация QR-кодов для быстрой настройки
- 🔗 Ссылки на конфигурацию для удобного доступа

## Установка

### Требования

- Python 3.10+
- SSH доступ к VPN серверу
- Docker с установленным AmneziaWG контейнером

### Шаги установки

1. **Клонируйте репозиторий или скопируйте файлы**

2. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Настройте переменные окружения:**
   
   Скопируйте `.env.example` в `.env` и заполните:
   ```env
   BOT_TOKEN=your_bot_token_here
   ADMIN_ID=your_telegram_id
   SERVER_HOST=194.26.27.31
   SERVER_USER=root
   SERVER_SSH_KEY=~/.ssh/id_ed25519
   VPN_PORT=40680
   VPN_DOCKER_CONTAINER=amnezia-awg
   ```

4. **Настройте SSH ключ:**
   
   Убедитесь, что у вас есть SSH ключ для доступа к серверу:
   ```bash
   ssh-keygen -t ed25519 -f ~/.ssh/id_ed25519
   ssh-copy-id root@194.26.27.31
   ```

5. **Создайте файл `contacts.json`:**
   
   Добавьте авторизованных пользователей:
   ```json
   {
     "+79001234567": {
       "telegram_id": 123456789,
       "username": "username",
       "first_name": "Имя",
       "last_name": "Фамилия"
     }
   }
   ```

6. **Инициализируйте базу данных:**
   
   База данных будет создана автоматически при первом запуске.

7. **Обновите БД для поддержки токенов (если БД уже существует):**
   ```bash
   python add_token_column.py
   ```

8. **Запустите веб-сервер для отдачи конфигураций по ссылке (в отдельном терминале):**
   ```bash
   python start_web_server.py
   ```
   Или запустите в фоновом режиме:
   ```bash
   nohup python start_web_server.py > web_server.log 2>&1 &
   ```

9. **Запустите бота:**
   ```bash
   python bot.py
   ```

## Использование

1. Откройте бота в Telegram: `@Moroz_VpnBot`
2. Отправьте команду `/start`
3. Если вы не авторизованы, нажмите "📱 Показать номер телефона"
4. После авторизации вы увидите главное меню:
   - 🔐 Получить AmneziaWG ключ
   - 📋 Мои ключи
   - ⚙️ Админ-панель (для администраторов)

## Создание VPN ключа

1. Нажмите "🔐 Получить AmneziaWG ключ"
2. Дождитесь создания ключа
3. Получите файл конфигурации (.conf) и QR-код
4. Импортируйте конфигурацию в приложение Amnezia VPN

## Управление ключами

- Просмотр всех ключей: "📋 Мои ключи"
- Удаление ключа: нажмите "🗑 Удалить" напротив нужного ключа

## Разработка

### Локальная разработка

Для локальной разработки:

1. Убедитесь, что SSH ключ настроен для доступа к серверу
2. Запустите бота локально:
   ```bash
   python bot.py
   ```

### Развертывание на сервере

После успешного тестирования локально:

1. Скопируйте файлы на сервер
2. Создайте systemd service файл
3. Запустите бота как сервис

## Безопасность

- ⚠️ Храните `.env` файл в безопасности
- ⚠️ Не публикуйте `BOT_TOKEN` и `ADMIN_ID`
- ⚠️ Используйте SSH ключи для доступа к серверу
- ⚠️ Регулярно обновляйте зависимости

## Известные проблемы

- Для локальной разработки требуется установленный `wg` (WireGuard tools) или SSH доступ к серверу
- При первом запуске может потребоваться подтверждение SSH ключа сервера
