## 2. МЕНЮ ПОКУПКИ QR-КОДОВ (НЕАВТОРИЗОВАННЫЕ ПОЛЬЗОВАТЕЛИ)

### 2.1. Интерфейс с кнопками +/-

**Дизайн меню:**
```
💳 Покупка QR-кодов

📅 Период подписки: [−] 1 месяц [+]

📦 Количество кодов: [−] 1 код [+]

💰 Сумма к оплате: 100₽

💡 Экономия при такой покупке:
   • Скидка за количество: 0% (0₽)
   • Скидка за период: 0% (0₽)
   • Общая экономия: 0₽

📝 Описание:
   Вы покупаете 1 QR-код на 1 месяц.
   После оплаты вы получите QR-код для настройки VPN.

[💳 Оплатить 100₽] [◀️ Назад]
```

**Логика работы:**
- По умолчанию: 1 месяц, 1 код
- Кнопки `[−]` и `[+]` для изменения значений
- Минимум: 1 месяц, 1 код
- Максимум: 12 месяцев, 5 кодов (или `QR_CODE_MAX_COUNT`)
- При изменении автоматически пересчитывается сумма
- Сумма отображается в кнопке "Оплатить"

### 2.2. Формула расчета цены (мультипликативная)

**Параметры:**
- `BASE_PRICE` = 100₽ (цена за 1 код на 1 месяц)
- `CODE_DISCOUNT` = 15% (скидка за каждый дополнительный код)
- `PERIOD_DISCOUNT`:
  - 1 месяц: 0%
  - 3 месяца: 5%
  - 6 месяцев: 10%
  - 12 месяцев: 20%

**Формула:**
```
Цена_за_код_за_месяц = BASE_PRICE * (1 - CODE_DISCOUNT * (n-1) / n) * (1 - PERIOD_DISCOUNT)

Итого = Цена_за_код_за_месяц * количество_кодов * количество_месяцев
```

**Примеры расчета:**
- 1 код, 1 месяц: 100 * (1 - 0) * (1 - 0) * 1 * 1 = 100₽
- 2 кода, 1 месяц: 100 * (1 - 0.15 * 1/2) * (1 - 0) * 2 * 1 = 185₽
- 1 код, 12 месяцев: 100 * (1 - 0) * (1 - 0.20) * 1 * 12 = 960₽
- 2 кода, 12 месяцев: 100 * (1 - 0.15 * 1/2) * (1 - 0.20) * 2 * 12 = 1,776₽

### 2.3. Отображение экономии

**Формат:**
```
💡 Экономия при такой покупке:
   • Скидка за количество: X% (Y₽)
   • Скидка за период: Z% (W₽)
   • Общая экономия: TOTAL₽
```

**Расчет экономии:**
- Экономия за количество = `BASE_PRICE * количество * месяцы * CODE_DISCOUNT * (n-1) / n`
- Экономия за период = `BASE_PRICE * количество * месяцы * PERIOD_DISCOUNT`
- Общая экономия = сумма всех скидок

### 3.2. Структура веб-панели

**URL структура:**
- `/admin/` - главная страница
- `/admin/users` - управление пользователями
- `/admin/traffic` - статистика трафика
- `/admin/payments` - платежи и статистика
- `/admin/settings` - настройки

**Авторизация:**
- Кнопка в админ-панели Telegram бота: "🌐 Открыть веб-панель"

- Токен хранится в сессии браузера

#### 3.3.1. Управление пользователями

## 4. СТАТИСТИКА ТРАФИКА

### 4.1. Получение статистики из WireGuard

**Метод 1: `wg show wg0 dump`**
```bash
wg show wg0 dump
```
Вывод содержит: public_key, endpoint, allowed_ips, latest_handshake, transfer (received, sent)

**Метод 2: `wg show wg0 transfer`**
```bash
wg show wg0 transfer
```
Вывод: public_key, transfer_bytes (received, sent)

**Рекомендация**: Использовать `wg show wg0 dump` для получения полной информации, включая последнее подключение.

### 4.2. Хранение статистики в БД

**Новая модель `TrafficStatistics`:**
```python
class TrafficStatistics(Base):
    """Статистика трафика по ключам"""
    __tablename__ = 'traffic_statistics'
    
    id = Column(Integer, primary_key=True)
    vpn_key_id = Column(Integer, ForeignKey('vpn_keys.id'), nullable=False)
    date = Column(Date, nullable=False)  # Дата статистики
    bytes_received = Column(BigInteger, default=0)  # Входящий трафик
    bytes_sent = Column(BigInteger, default=0)  # Исходящий трафик
    connection_ips = Column(String(1000), nullable=True)  # JSON массив IP адресов
    last_connection = Column(DateTime, nullable=True)  # Последнее подключение
    created_at = Column(DateTime, default=datetime.now)
    
    # Связь с ключом
    vpn_key = relationship("VPNKey", backref="traffic_stats")
```

**Обновление статистики:**
- При загрузке страницы `/admin/traffic` выполняется запрос к FastAPI
- FastAPI получает статистику из WireGuard через `vpn_manager`
- Данные сохраняются в БД (если нужно хранить историю) или только отображаются

**IP адреса подключений:**
- Хранить последние 5-10 IP адресов для каждого ключа
- Обновлять при каждом получении статистики
- Формат: JSON массив `["192.168.1.1", "10.0.0.1", ...]`

### 4.3. Период статистики

**По умолчанию:** Текущий календарный месяц

**Выбор периода:**
- Выпадающий список: "Текущий месяц", "Прошлый месяц", "Последние 30 дней", "Выбрать период"
- При выборе "Выбрать период" - календарь для выбора дат
