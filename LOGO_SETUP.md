# Настройка логотипа бота

## Способ 1: Установка фото профиля бота (через BotFather)

1. Откройте [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте команду `/setuserpic`
3. Выберите вашего бота из списка
4. Отправьте изображение логотипа (PNG или JPEG, квадратное, до 640x640px)

## Способ 2: Использование логотипа в сообщениях бота

1. Сохраните логотип как `logo.png` в директории бота (`/root/logo.png` на сервере)
2. Логотип будет автоматически отправляться в приветственном сообщении `/start`

### Требования к логотипу:
- Формат: PNG или JPEG
- Рекомендуемый размер: 512x512px или 1024x1024px (квадратное изображение)
- Максимальный размер файла: 10 МБ

### Путь к логотипу:
- По умолчанию: `logo.png` в корневой директории бота
- Можно изменить через переменную окружения `BOT_LOGO_PATH` в `.env`

### Пример настройки в .env:
```
BOT_LOGO_PATH=/root/logo.png
```

## Загрузка логотипа на сервер

```bash
# С локального компьютера
scp logo.png root@194.26.27.31:/root/

# Или через SSH
ssh root@194.26.27.31
# Затем загрузите файл через scp или другой способ
```

## Проверка

После загрузки логотипа перезапустите бота:
```bash
ssh root@194.26.27.31 "systemctl restart vpn-bot"
```

Затем отправьте команду `/start` боту - логотип должен отобразиться в приветственном сообщении.
