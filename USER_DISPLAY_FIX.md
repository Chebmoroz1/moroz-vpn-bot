# Исправление отображения пользователей

## Проблема

Пользователь с номером телефона +79933393296 не отображался ни в веб-панели, ни в боте.

## Причина

Пользователь был помечен как удаленный (`is_deleted = True`), поэтому фильтры в веб-панели и боте его не показывали.

## Решение

### 1. Восстановление пользователя

Пользователь восстановлен в БД:
- `is_deleted = False`
- `is_active = True`

### 2. Убрана зависимость от contacts.json

Функция `_handle_admin_add_user_text` теперь работает только с БД:
- При добавлении по номеру телефона - создается/восстанавливается пользователь в БД
- При добавлении по Telegram ID - создается/восстанавливается пользователь в БД
- Убраны все обращения к `contacts.json`

### 3. Единая логика работы

Теперь вся система работает через БД:
- ✅ Авторизация - проверка по БД
- ✅ Добавление пользователей - только в БД
- ✅ Отображение пользователей - фильтр по `is_deleted == False`
- ✅ Удаление пользователей - hard delete (полное удаление из БД)

## Проверка

Пользователь с номером +79933393296 теперь должен отображаться:
- ✅ В веб-панели в списке "Пользователи"
- ✅ В боте в меню "👥 Управление пользователями"
