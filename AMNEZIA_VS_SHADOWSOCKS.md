# Amnezia VPN vs ShadowSocks - Решение проблемы

## Проблема

Amnezia VPN приложение не принимает стандартные WireGuard конфигурации, потому что:
- AmneziaWG использует свой собственный формат конфигурации
- Требуется установка Amnezia VPN на сервере через их клиент
- Конфигурации должны быть созданы через Amnezia VPN клиент

## Решение: Используйте ShadowSocks

**ShadowSocks уже настроен и работает на сервере!**

### Преимущества ShadowSocks для России:

1. ✅ **Уже настроен** - работает на порту 8388
2. ✅ **Лучше обходит блокировки** - специально разработан для обхода цензуры
3. ✅ **Стандартный формат** - работает с любыми ShadowSocks клиентами
4. ✅ **Не требует специального приложения** - множество клиентов на выбор

### Как использовать ShadowSocks:

1. **Получите ключ через бота:**
   - Откройте бота @Moroz_VpnBot
   - Нажмите "🔒 ShadowSocks (для России)"
   - Получите конфигурацию и QR-код

2. **Установите клиент ShadowSocks:**
   - **Android:** Shadowsocks, Outline, Shadowsocks Android
   - **iOS:** Outline, Shadowrocket (платное)
   - **Windows/Mac:** Shadowsocks-Windows, Outline Client

3. **Импортируйте конфигурацию:**
   - Отсканируйте QR-код или импортируйте JSON файл
   - Подключитесь

## Альтернатива: Настройка Amnezia VPN

Если все же нужно использовать Amnezia VPN:

1. **Установите Amnezia VPN клиент** на ваш компьютер
2. **Подключитесь к серверу** через SSH
3. **Установите VPN** через Amnezia клиент
4. **Экспортируйте конфигурацию** для AmneziaWG

Но это сложнее и требует установки Amnezia VPN на сервере.

## Рекомендация

**Используйте ShadowSocks** - он уже настроен, работает лучше в России и проще в использовании!
