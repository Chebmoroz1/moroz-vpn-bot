# Установка AmneziaWG на сервере

## Проблема

ShadowSocks не работает, а предыдущий сервер использовал AmneziaWG, который отлично работает в России.

## Решение: Установка AmneziaWG

AmneziaWG - это форк WireGuard с обфускацией, специально разработанный для обхода блокировок в России.

### Вариант 1: Установка через Amnezia VPN клиент (рекомендуется)

1. **Скачайте Amnezia VPN клиент:**
   - Windows/Mac/Linux: https://github.com/amnezia-vpn/amnezia-client/releases
   - Android/iOS: App Store / Google Play

2. **Подключитесь к серверу:**
   - Откройте Amnezia VPN клиент
   - Выберите "Self-hosted VPN"
   - Введите данные сервера:
     - IP: 194.26.27.31
     - Пользователь: root
     - Используйте SSH ключ для подключения

3. **Установите AmneziaWG:**
   - Выберите протокол "AmneziaWG"
   - Дождитесь завершения установки
   - Экспортируйте конфигурацию для клиентов

### Вариант 2: Ручная установка (сложнее)

AmneziaWG можно установить вручную, но это требует больше настроек.

## Текущий статус

- WireGuard установлен и работает
- ShadowSocks установлен, но не работает
- AmneziaWG не установлен

## Рекомендация

Используйте **Вариант 1** - установите Amnezia VPN через их клиент. Это самый простой и надежный способ.

После установки AmneziaWG через клиент, можно будет:
1. Экспортировать конфигурации для клиентов
2. Интегрировать в бота для автоматической генерации
3. Использовать AmneziaWG, который отлично работает в России
