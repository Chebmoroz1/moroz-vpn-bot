1. **Очистите кэш браузера:**

   - Firefox: Ctrl+F5 (Windows) или Cmd+Shift+R (Mac)
   - Safari: Cmd+Option+R

2. **Проверьте консоль браузера (F12):**
   - Убедитесь, что нет ошибок загрузки JS файлов
   - Проверьте, что запросы к API проходят успешно

3. **Проверьте, что frontend собран:**
   ```bash
   ssh root@194.26.27.31 'ls -la /opt/vpn-bot/web_admin/frontend/build/static/js/'
   ```

4. **Проверьте nginx:**
   ```bash
   ssh root@194.26.27.31 'sudo nginx -t && sudo systemctl status nginx'
   ```
