# Руководство по тестированию веб-панели

## Быстрый старт

### 1. Автоматическое тестирование API

Для автоматического тестирования всех API endpoints:

```bash
# Убедитесь, что FastAPI backend запущен
# Локально: python3 web_admin/backend/main.py
# Или на сервере: systemctl status vpn-web-admin.service

# Запустите тесты
python3 test_web_admin_api.py
```

Для тестирования на удаленном сервере:

```bash
WEB_ADMIN_API_URL=http://194.26.27.31:8889 python3 test_web_admin_api.py
```

### 2. Ручное тестирование через браузер

1. **Получите токен через Telegram бота:**
   - Откройте бота
   - Перейдите в "⚙️ Админ-панель"
   - Нажмите "🌐 Веб-панель"
   - Скопируйте ссылку с токеном

2. **Откройте веб-панель:**
   - Перейдите по ссылке или откройте `http://194.26.27.31:8889/?token=YOUR_TOKEN`
   - Проверьте, что авторизация прошла успешно

3. **Протестируйте функции:**
   - Dashboard - проверьте статистику
   - Пользователи - создайте, отредактируйте, удалите пользователя
   - Ключи - просмотрите, активируйте/деактивируйте ключи
   - Трафик - проверьте статистику трафика

## Устранение проблем

### Тесты не проходят

1. **Проверьте, что backend запущен:**
   ```bash
   # Локально
   curl http://localhost:8889/
   
   # На сервере
   curl http://194.26.27.31:8889/
   ```

2. **Проверьте логи:**
   ```bash
   # На сервере
   ssh root@194.26.27.31 'sudo journalctl -u vpn-web-admin.service -n 50'
   ```

3. **Проверьте, что администратор существует в БД:**
   ```bash
   python3 -c "from database import SessionLocal, User; from config import ADMIN_ID; db = SessionLocal(); admin = db.query(User).filter(User.telegram_id == ADMIN_ID).first(); print('Admin found:', admin is not None); db.close()"
   ```

### Ошибка "Token not provided"

- Убедитесь, что токен передается в параметрах запроса
- Проверьте, что токен не истек (действителен 24 часа)
- Получите новый токен через Telegram бота

### Ошибка "Access denied"

- Убедитесь, что пользователь является администратором
- Проверьте `ADMIN_ID` в `.env` файле
- Убедитесь, что пользователь существует в БД с `is_admin=True`

## Результаты тестирования

После запуска автоматических тестов вы увидите:
- Количество пройденных тестов

- Количество пропущенных тестов
- Процент успешности

```
✗ Провалено: 0
⚠ Пропущено: 2

Успешность: 83.3%
============================================================
```
