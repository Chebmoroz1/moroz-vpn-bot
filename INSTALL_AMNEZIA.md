# Установка Amnezia VPN (AmneziaWG) на сервере

## Важно

Amnezia VPN обычно устанавливается через их клиент приложение. Это самый простой и надежный способ.

## Шаг 1: Установка Amnezia VPN клиента

1. **Скачайте Amnezia VPN клиент:**
   - Windows/Mac/Linux: https://github.com/amnezia-vpn/amnezia-client/releases
   - Или с официального сайта: https://amnezia.org/

2. **Установите клиент на ваш компьютер**

## Шаг 2: Подключение к серверу

1. **Откройте Amnezia VPN клиент**
2. **Выберите "Self-hosted VPN"** (Собственный VPN сервер)
3. **Введите данные сервера:**
   - **IP адрес:** 194.26.27.31
   - **Пользователь:** root
   - **Метод подключения:** SSH ключ
   - **SSH ключ:** Используйте ваш ключ `~/.ssh/id_ed25519`

## Шаг 3: Установка AmneziaWG

1. **После подключения к серверу:**
   - Выберите протокол **"AmneziaWG"**
   - Нажмите "Установить"
   - Дождитесь завершения установки

2. **После установки:**
   - AmneziaWG будет установлен на сервере
   - Вы сможете создавать конфигурации для клиентов
   - Конфигурации можно экспортировать в формате AmneziaWG

## Шаг 4: Интеграция с ботом

После установки AmneziaWG через клиент, можно будет:
1. Найти конфигурационные файлы на сервере
2. Интегрировать генерацию конфигураций в бота
3. Автоматически генерировать ключи для пользователей

## Альтернатива: Ручная установка

Если нужно установить без клиента, можно попробовать:
- Использовать Docker контейнеры Amnezia
- Установить компоненты вручную

Но это сложнее и требует больше настроек.
