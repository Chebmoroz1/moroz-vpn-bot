## 📋 Быстрый деплой

### Вариант 1: Автоматический (рекомендуется)

```bash
# Просто запустите скрипт деплоя
./deploy/deploy.sh
```

Скрипт автоматически:
- Соберет frontend
- Скопирует файлы на сервер
- Установит зависимости
- Перезапустит сервисы

### Вариант 2: Ручной

```bash
# 1. Соберите frontend локально
cd web_admin/frontend
npm install
npm run build
cd ../..

# 2. Скопируйте файлы на сервер
scp -r web_admin root@194.26.27.31:/opt/vpn-bot/

# 3. Соберите frontend на сервере (если не собрали локально)
ssh root@194.26.27.31 'cd /opt/vpn-bot/web_admin/frontend && npm install && npm run build'

# 4. Перезапустите сервис
ssh root@194.26.27.31 'sudo systemctl restart vpn-web-admin.service'
```

## 🔍 Проверка после деплоя

### 1. Проверка статуса сервиса

```bash
ssh root@194.26.27.31 'sudo systemctl status vpn-web-admin.service'
```

Должен быть статус: `active (running)`

### 2. Проверка frontend

```bash
ssh root@194.26.27.31 'ls -la /opt/vpn-bot/web_admin/frontend/build/'
```

Должна существовать директория `build` с файлами `index.html`, `static/` и т.д.

### 3. Проверка в браузере

1. Откройте Telegram бота
2. Перейдите в "⚙️ Админ-панель" → "🌐 Веб-панель"
3. Откройте ссылку с токеном
4. Проверьте наличие новых функций:
   - На странице "Пользователи": кнопки "📥 Экспорт CSV", "📤 Импорт CSV", "✏️"
   - На странице "Ключи": кнопки управления ключами

## ⚠️ Важные замечания

1. **Node.js на сервере** - если Node.js не установлен на сервере, frontend не соберется. Установите:
   ```bash
   ssh root@194.26.27.31 'curl -fsSL https://deb.nodesource.com/setup_18.x | bash - && apt-get install -y nodejs'
   ```

2. **Кэш браузера** - после деплоя очистите кэш браузера (Ctrl+Shift+R или Cmd+Shift+R)

3. **Токены** - токены авторизации хранятся в памяти FastAPI, при перезапуске сервиса все токены станут недействительными

4. **База данных** - при деплое база данных не затрагивается, все данные сохраняются
