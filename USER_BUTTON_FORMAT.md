# Форматирование кнопок пользователей в админ-панели

## Описание

Улучшен интерфейс списка пользователей в админ-панели. Теперь каждая кнопка пользователя отображает информацию о количестве ключей в формате `(выданных/разрешенных)`.

## Формат кнопки

```
👤 Имя пользователя (X/Y)
```

Где:
- `👤` - иконка пользователя
- `Имя пользователя` - отображаемое имя (может быть обрезано)
- `X` - количество выданных активных ключей
- `Y` - максимальное количество разрешенных ключей (`max_keys`)

## Примеры

### Короткое имя
```
👤 Иван (2/5)
```

### Длинное имя (обрезается)
```
👤 ОченьДлинноеИмяПользователяКоторое... (3/10)
```

### С username
```
👤 Никнейм (@username) (1/3)
```

## Логика обрезки

Функция `_format_user_button_name()` умно обрезает длинные имена:

1. **Резервирует место** для префикса `👤 ` (2 символа) и суффикса ` (X/Y)` (6-10 символов)
2. **Вычисляет доступную длину** для имени: `max_length - reserved_length`
3. **Обрезает имя** если оно длиннее доступной длины, добавляя `...`
4. **Гарантирует минимум** 5 символов для имени (чтобы было хоть что-то видно)

### Параметры

- `max_length = 50` - максимальная длина кнопки (по умолчанию)
- Минимальная длина имени: 5 символов
- Многоточие: 3 символа

### Примеры обрезки

| Исходное имя | Длина | Результат |
|--------------|-------|-----------|
| "Иван" | 4 | `👤 Иван (2/5)` |
| "ОченьДлинноеИмя" | 15 | `👤 ОченьДлинноеИмя (1/3)` |
| "ОченьОченьОченьДлинноеИмяПользователя" | 35 | `👤 ОченьОченьОченьДлинноеИмяПольз... (5/10)` |

## Использование

Функция вызывается в `_handle_admin_users()` при создании кнопок:

```python
button_text = self._format_user_button_name(display_name, keys_count, user.max_keys)
keyboard.append([
    InlineKeyboardButton(button_text, callback_data=f"admin_user_detail:{user.id}")
])
```

## Преимущества

✅ **Информативность** - сразу видно количество ключей у каждого пользователя  
✅ **Компактность** - информация в одной строке  
✅ **Читаемость** - длинные имена обрезаются, но информация о ключах всегда видна  
✅ **Удобство** - не нужно открывать детали пользователя, чтобы узнать количество ключей
